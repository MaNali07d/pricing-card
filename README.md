**Artisiannest Pricing Card**
---
A clean and minimal pricing card website for Artisiannest, showcasing handmade gift categories and pricing details.

---
**Live Website**
---
https://artisiannest-pricing-card.vercel.app/
---
Built With

HTML

Tailwind CSS (via CDN)

This is a static website. No frameworks, no build tools, and no JavaScript libraries are used.

---
**Features**

Mobile-first responsive layout

Clear pricing sections for small, medium, and large gifts

Customization and offer details

Simple and elegant UI
